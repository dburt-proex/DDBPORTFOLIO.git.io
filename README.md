# Drew Burt Portfolio

**Canonical public portfolio:** [drew-burt-portfolio.dburt30.chatgpt.site](https://drew-burt-portfolio.dburt30.chatgpt.site)

This repository is the versioned documentation and evidence layer for Drew Burt's public portfolio. It records the claims, proof links, maturity boundaries, and publishing rules behind the public experience.

## Source-of-truth model

| Scope | Canonical authority | Purpose |
| --- | --- | --- |
| Public positioning, navigation, assessment, lab, and contact paths | [ChatGPT Sites portfolio](https://drew-burt-portfolio.dburt30.chatgpt.site) | The current visitor-facing experience |
| Portfolio evidence map, maturity boundaries, and change history | This repository | Versioned documentation and review |

The two portfolio builds are not maintained as competing websites. The GitHub Pages root hands visitors to the Sites portfolio. Existing technical briefs remain available as supporting evidence.

## Publishing rule

1. Verify a technical claim in the relevant project repository.
2. Update this repository when the evidence map, maturity boundary, or public claim changes.
3. Update ChatGPT Sites when visitor-facing wording or functionality changes.
4. Check that public links resolve to the documented evidence.
5. Record the change through Git history or a pull request.

There is no automatic two-way synchronization. That is intentional: evidence must be reviewed before it becomes a public claim.
